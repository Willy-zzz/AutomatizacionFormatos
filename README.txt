GENERADOR DE FORMATOS FVS
Versión 1.0 - por Willys Arias

REQUISITOS:
- Microsoft Word instalado
- Archivo Excel con los datos descargado del SAP
- Plantilla Word con campos {{llave}} correctamente escritos

PASOS PARA USO:
(El ejecutable final estará en: dist/GeneradorFVS.exe)
1. Abrir el programa (doble clic en el ejecutable)
2. Cargar el archivo Excel cuando se solicite
3. Seleccionar un ID / Posición del menú desplegable
4. Se generará un documento Word con los datos llenados
5. Elige dónde quieres guardar el documento generado

NOTAS:
- Puedes usar tu propia plantilla .docx. Asegúrate de que las llaves coincidan con los nombres.
- Si necesitas cambiar el Excel, solo vuelve a cargar otro desde el botón de selección.


DESCRIPCIÓN:
Este generador automatiza el llenado de formatos Word (FVS) a partir de información contenida en archivos Excel descargados del sistema SAP. Facilita la creación de documentos oficiales de recursos humanos como solicitudes de movimiento de plazas, validación de candidatos y más.

FUNCIONALIDADES:
- Lectura automática de archivos Excel con doble encabezado
- Selección rápida de registros mediante "ID / Posición"
- Relleno inteligente de plantillas Word con llaves como {{Nombre}}, {{Categoria}}, etc.
- Soporte para múltiples campos, incluidos los del candidato seleccionado
- Generación de documentos Word limpios y listos para impresión o envío
- Elección libre del nombre y carpeta de guardado final
- Compatibilidad con múltiples versiones de Word

OBJETIVO:
Reducir el tiempo y los errores humanos al generar formatos institucionales repetitivos, y brindar una solución práctica que funcione completamente en local, sin requerir conexión a internet ni instalación adicional.

FUTURAS MEJORAS:
- Soporte para múltiples plantillas seleccionables
- Gestión de plantilla personalizada (subir, listar, eliminar)
- Previsualización de datos antes de generar el documento
- Exportación masiva por lotes
- Integración con firmas digitales (opcional)
- Versión web con acceso restringido
